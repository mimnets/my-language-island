## 🛠️ Monirul's Project Blueprint: AI Language Island

এই প্রজেক্টটি মূলত একটি **"Full-stack Serverless Application"**। এটি ৩টি স্তরে কাজ করে:

### ১. ডেটা লেয়ার (Data Layer) - GitHub

* **কেন:** এটি আপনার ডাটাবেজ হিসেবে কাজ করে। ফ্রিতে ডাটা সেভ রাখার জন্য গিটহাব সেরা।
* **ফাইল:** `data.json` (একটি সাধারণ টেক্সট ফাইল যা JSON ফরম্যাটে ডাটা রাখে)।

### ২. ব্যাকএন্ড লেয়ার (Backend/API Layer) - Vercel Serverless Functions

* **কেন:** আপনার ব্রাউজার (Frontend) সরাসরি গিটহাবের সাথে কথা বলতে পারে না নিরাপত্তার কারণে। তাই মাঝখানে একটি 'সার্ভার' লাগে।
* **ফাইল:** `api/index.js` (Express.js ব্যবহার করে তৈরি)। এটি গিটহাবের **REST API** ব্যবহার করে ডাটা আনে (GET) এবং জমা দেয় (PUT)।
* **সিকিউরিটি:** `GITHUB_TOKEN` এখানে ব্যবহার করা হয় যাতে আপনার পারমিশন ছাড়া কেউ ডাটা পরিবর্তন করতে না পারে।

### ৩. ফ্রন্টএন্ড লেয়ার (Frontend/UI Layer) - HTML/Tailwind

* **কেন:** ইউজার ইন্টারফেস বা ড্যাশবোর্ড যা আপনি চোখে দেখেন।
* **ফাইল:** `public/index.html`। এটি জাভাস্ক্রিপ্টের `fetch()` মেথড ব্যবহার করে আপনার তৈরি করা API-কে কল করে।

## 🚀 নতুন প্রজেক্ট করার ৭টি ধাপ (The Master Checklist)

আপনি যখন ভবিষ্যতে নতুন কোনো প্রজেক্ট একা করবেন, তখন নিচের ধাপগুলো অনুসরণ করবেন:

1. **Repository Setup:** গিটহাবে নতুন রিপোজিটরি খুলে একটি ব্ল্যাঙ্ক ডাটা ফাইল (`.json`) তৈরি করা।
2. **Token Generation:** গিটহাব থেকে টোকেন নেওয়া (অবশ্যই **Write Permission** সহ)।
3. **Local Coding:** আপনার কম্পিউটারে Node.js প্রজেক্ট শুরু করা (`npm init`) এবং `express` ও `axios` ইন্সটল করা।
4. **API Development:** গিটহাব এপিআই-এর সাথে কানেক্ট করার জন্য ব্যাকএন্ড কোড লেখা।
5. **UI Design:** ইউজারের জন্য ফর্ম এবং লিস্ট দেখার ইন্টারফেস বানানো।
6. **Vercel Deployment:** গিটহাবের সাথে ভার্সেল কানেক্ট করা।
7. **Environment Variables:** ভার্সেলের সেটিংসে গিয়ে সিক্রেট টোকেন (Token) যুক্ত করা।
